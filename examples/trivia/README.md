# Trivia - Juego de Preguntas 6502

Un juego de preguntas y respuestas para el Monitor 6502.

## Descripción

Pon a prueba tus conocimientos de informática con 5 preguntas de opción múltiple.

## Cómo Jugar

1. Carga el programa en el monitor
2. Lee cada pregunta
3. Escribe la letra de tu respuesta (A, B, C o D)
4. Al final verás tu puntuación

## Preguntas

El juego incluye preguntas sobre:
- Computadoras históricas
- Microprocesadores
- Lenguajes de programación
- Historia de la informática

## Compilación

```bash
make clean
make
```

## Uso

```
LOAD TRIVIA
G 0800
```

## Características

- 5 preguntas de cultura informática
- Sistema de puntuación
- Retroalimentación inmediata
- Retorna al monitor al terminar

## Hardware Requerido

- Tang Nano 9K con Monitor 6502
- Conexión UART para terminal
